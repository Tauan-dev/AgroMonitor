# Agricultural Planning Analysis System

Aplicação full-stack para análise numérica de planejamento agrícola usando álgebra linear, sensibilidade e visualização.

## Arquitetura

- **Backend**: FastAPI (Python) com algoritmos de álgebra linear
- **Frontend**: Angular com UI/UX moderna
- **Containerização**: Docker & Docker Compose

## Pré-requisitos

- Docker >= 20.10
- Docker Compose >= 2.0
- Node.js >= 18 (somente se desenvolver localmente)
